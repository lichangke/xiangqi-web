# 象棋网页版项目（暂定）

## 项目说明
这是一个以中国象棋为核心的 Web 项目目录，当前处于 SDD 的需求讨论阶段。

本项目第一轮目标聚焦：
- 严格规则的人机对战
- AI 棋子人格与话剧式讨论演绎
- 支持电脑端与手机端
- 普通用户登录、后台管理、模型配置
- 为未来人人对战与 Qt 桌面客户端预留扩展空间
